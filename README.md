# Plarza

#### Dead simple social media aggregator

### Run with Docker 
```bash
docker compose up
```

## Client

**Install**

```bash
$ cd client
$ npm i
```

**Run**

```bash
$ npm run dev -- --open
```

## Server

**Install**

```bash
$ pip install -r requirements.txt
```

**Run**

```bash
flask run --debug
```

## Database

```bash
psql -U postgres
\c testdb
```
